## データ構造

各シートには以下の構造でデータが格納されています：

### 時間軸の表現
- 列ヘッダー: 191^1-6100, 291^1-6100, 391^1-6100, 491^1-6100, 591^1-6100, 691^1-6100
- 数字は時間足を表す（1=1分, 2=5分, 3=15分, 4=30分, 5=60分, 6=240分）

### 値の意味
- 0: 条件不成立/無効
- 1: 上方向/買い条件成立
- 2: 下方向/売り条件成立
- 3: 待機/ニュートラル

## 使用例（強追加シートより）

```
Time                  191^1-6100  191^1-6101  191^1-6102  ...  強追加_1  強追加_5  強追加_15  強追加_30  強追加_60  強追加_240
2019-10-07           0           0           3           ...  0         0         0          0          0          2
2019-10-07 00:01:00  0           0           3           ...  0         0         0          0          0          0
```

この例では：
- 191^1-6102 = 3：1分足で待機状態
- 強追加_240 = 2：240分足で下方向の強追加シグナル
